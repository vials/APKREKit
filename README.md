Yet another Android based script, APKReKit (APK Reverse Engineering Kit). *ARK* for short

Utilizes Python3, Java, and apktool

Downloads:
https://apktool.org/
https://www.java.com/en/download/manual.jsp
https://www.python.org/

This script does as described: Decompiles, decompresses and uses regex to search for exposed endpoints, and other data. 
Very useful for finding hidden endpoints and bug research.
